# ⚽ EPL Prediction System

A Streamlit-based football prediction system using Monte Carlo simulation.

## Features
- League table simulation
- Live results integration (API)
- Live score support
- Monte Carlo predictions
- Relegation & Top 5 probability

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy

## How to Run

```bash
pip install -r requirements.txt
streamlit run english_premier_league.py

## Notes

API key required for live data
Falls back to model if API unavailable
